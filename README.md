# 맥북 첫걸음 💻

맥북을 처음 사용하는 초보자를 위한 한국어 기초 사용법 교육 웹앱입니다.

**바로 사용하기 → https://macbook-guide-steel.vercel.app**

## 구성

10개의 짧은 레슨과 최종 퀴즈로 이루어져 있습니다.

1. 처음 켜기 & 기본 설정 — 전원, 초기 설정, Apple ID
2. 트랙패드 제스처 — 손가락 개수별 8가지 제스처
3. 키보드 & 단축키 — ⌘ 키와 필수 단축키, 실전 연습 트레이너
4. 화면 구성 이해하기 — 메뉴 막대, Dock, Finder, Spotlight
5. 창 관리 & 멀티태스킹 — 신호등 버튼, Split View, 미션 컨트롤
6. 일상 활용 — 앱 설치·삭제, 스크린샷, 아이폰 연동
7. 기본 앱 둘러보기 — Safari, 미리보기, 메모와 국민 앱
8. 윈도우 사용자라면 — 윈도우 ↔ 맥 차이 비교표
9. 백업·보안·업데이트 — Time Machine, iCloud, 나의 찾기
10. 자주 묻는 문제 해결 — 초보자 단골 질문 FAQ

## 특징

- 프레임워크 없는 **단일 HTML 파일** — 브라우저에서 바로 열립니다
- 레슨 완료·체크리스트·진행률이 localStorage에 자동 저장
- 실제 키 입력을 감지하는 단축키 연습 트레이너
- 문제마다 해설이 붙은 10문제 최종 퀴즈
- 모바일 반응형 지원

## 로컬에서 열기

```bash
open index.html
```

또는 정적 서버로:

```bash
npx serve .
```

## 배포

[Vercel](https://vercel.com)로 배포되어 있습니다.

```bash
vercel deploy --prod
```
